Instructions given for the markup in Html lengauge referenced as Tags

Types of Tags

  1. Paired Tags

    Tags needs closing tag as well are called paired tags, e.g p, div, headings etc

  2. Slef Closing tags

    Tags that doesn't require closing tags are self closing tags, e.g br, hr etc

Symantic and non-symantic tags

  1. Symantic Tags

    The tag explains it's contents by it's name is a symantic tag, like header, article

  2. Non Symantic Tags

    The tag which not explains it's contents is non-symantic tag, like div, span

Block Level Elements and Inline Elements

  1. Block Level Element

    These tags by default aquire all the width and the next tag will start from the next line, p, div

  2. Inline Level Element

    These tags by default aquire just the width necessry for it's contents and the next tag will start right after them, like img, a, button, span etc
