CSS

Type of CSS

  • Inline CSS
    We can style into the tag itself by using it's style attribute, this approach is called Inline CSS.

    For Example: we applied color to a paragraph using style attribute by inline method

                    <p style="color: red"> Hello World! </p>

    Output:

    Hello World!

  • Internal CSS
    We can style into the style tag in same file, this approach is called Internal CSS.

    For Example: we applied background-color to a span using style tag by Internal method.

                    <style>
                    span {
                        background-color: green;
                        color: white;
                    }
                </style>
            
                <span>Hy, my name is John.</span>

    Output:

    Hy, my name is John.
  • External CSS
    In this mechanism we create another css file and than attach it to the document by <link /> tag.

    For Example: we applied heartbeat animation to the pre tag by External CSS.

                    <link rel="stylesheet" href="./style.css">
